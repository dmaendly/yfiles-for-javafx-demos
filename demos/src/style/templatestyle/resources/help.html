This demo shows how to use TemplateNodeStyle, TemplateLabelStyle and TemplatePortStyle to create complex node and label visualizations using FXML components.

The yFiles for JavaFX template styles allow to use arbitrary FXML components for graph item visualization, while being able to access all relevant context information from the component implementation. Most context properties can be used with data binding.

This demo uses so-called "user-tags" to associate complex custom data objects with graph items.

Things to try

  • Select nodes or edge labels to see custom selection highlighting defined in the node and label templates.
  • Create new customers styled using the TemplateNodeStyle by clicking on the background.
  • Create new relations between customers and products with the template label visualization. Note that only edge with direction from customers to products are allowed.
