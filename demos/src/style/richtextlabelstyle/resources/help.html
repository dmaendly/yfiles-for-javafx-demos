JavaFX 8 provides an API for rich text styling using TextFlows. yFiles styles offer the ability to easily adapt those API to display rich text labels. We use basic elements of Markdown to define how a label should look.

Markdown rules in this demo:

  • An empty line creates a paragraph.
  • Two spaces at the end of a line create a line break.
  • Text enclosed by a pair of single asterisks or underscores is displayed italic.
  • Text enclosed by a pair of double asterisks or underscores is displayed bold.
  • Text enclosed by a pair of triple asterisks or underscores is displayed italic and bold.
  • Those rules can be nested and span over line breaks.

Things to try

  • Select a node or edge and hit F2 to edit the label in Markdown. Hit Enter to see the Markdown formatted label.
