This demo shows how to style the built-in templates and UI elements of the yFiles for JavaFX library with CSS to match your color theme.

Most visual elements like snaplines, indicators, handles, port and label candidates etc. have built-in style classes and can be styled out of the box with CSS.

However, the included visualizations of the graph elements do not support CSS. Therefore, this demo offers extra styles with CSS support.

Things to try

  • Choose a new theme from the drop-down in the toolbar.
  • Use the toolbar button Show CSS to open a text area showing the current CSS code.
  • Change the CSS code and press the Reload button to apply the changes.
