Step 1: Creating the View

The class GraphControl is the central UI element for working with graphs.

This application does not provide any interactive features except mouse wheel scrolling which is enabled by default. It is merely meant as a very basic example of how to display a GraphControl within a JavaFX application.

See the sources for details.
